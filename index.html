This project was built by Margaret ONeill, Matt Raines, and Jonathon Ray. It allows a user to find breweries in a given location. The initial source of information is Open Brewery DB. We also use Open Cage API to get lattitude and longitude location information. Finally the Yelp API is used for detailed brewery information, ratings, and photos.
